1- Baixe o arduino ide, eu prefiro a versão legacy

2- adicione em file > preference o link para baixar a placa esp32

https://espressif.github.io/arduino-esp32/package_esp32_index.json

<img width="1383" height="784" alt="image" src="https://github.com/user-attachments/assets/20427399-4104-499f-b1fb-90011259b19f" />

depois em placa > gerenciador de placas e baixe a placa: esp32 by expressif

Selecione a placa:

<img width="1082" height="638" alt="image" src="https://github.com/user-attachments/assets/3c9dd506-e2d5-4334-afb3-003e0a3ab24d" />

Baixe a wifi manager (biblioteca necessária para colocar wifi no projeto)

https://github.com/tzapu/WiFiManager/archive/refs/tags/v2.0.17.zip

Depois vá em.

Sketch > Add biblioteca de .zip.

Por fim passe o código pra placa, selecione a porta com que apareceu depois que colocou a plaquinha no cabo usb.

segure o botão de boot, toque e solte no reset, mantenha o botão de boot segurado e clique na setinha upload para passar o código pra placa.
